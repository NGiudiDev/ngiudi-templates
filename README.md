# ngiudi-templates

Este repositorio contiene diferentes templates para proyectos backend y frontend, facilitando la creación rápida de aplicaciones con buenas prácticas y estructuras modernas.

## Templates disponibles

- **node-crud-template**: Template para un sistema backend CRUD usando Node.js, Express y Sequelize, aplicando Clean Architecture. Permite crear APIs genéricas para cualquier tabla de base de datos.

- **storybook-tempalte**: Template para iniciar un entorno de Storybook con Vite y React. Incluye configuración básica, ejemplos de stories y scripts para desarrollo y build de componentes UI de forma aislada. Ideal para documentar y testear componentes visuales en proyectos frontend.

Próximamente se agregarán más templates para otros stacks y arquitecturas.
